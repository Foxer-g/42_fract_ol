# FRACT'OL

This project is about generating fractals and
learn graphical display management with the 42\_MiniLibX also known as "**MLX**".

## INPUTS
fract'ol is only develloped to generate :
 - Mandelbrot
> ```./fractol mandelbrot```
 - Julia
> ```./fractol julia <Param_1> <Param_2>```

## OUTPUT

The output is generated as a window through the **MLX**

Julia :

## PROGRAM INTERACTION :

- To escape the program you can press **\<Esc\>**
- The zoom is localized on the **scroll up** and **scroll down**
